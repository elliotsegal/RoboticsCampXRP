# Robot Code

Instead of using one of the templates or examples, we're going to start with a project that I prepared. It's very similar to the "XRPReference" project we used last week, but I've made some changes to help keep things organized.

## Getting the code

1. Go to <https://github.com/elliotsegal/RoboticsCampXRP>
2. Click the green "Code" button, then click "Download ZIP"
3. After the file downloads, extract it somewhere
4. Open 2026 WPILib VS Code
5. In the "File" menu of VS Code, select "Open Folder...", and navigate to the extracted project

## Using last week's code

You may want to refer to last week's code, or use it in this project. There are two ways to do that: you can open individual files from other projects (File > Open File...), or you can open a new VS Code window and have two projects open at once (File > New Window, then File > Open Recent in the new window).
